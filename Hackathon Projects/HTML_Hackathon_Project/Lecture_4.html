Filepath System :

  • File path tells the browser where a file is located (image, CSS, JS, etc.).

  • Two types:

Example Folder Structure :


1. Absolute Path :

ðŸ‘‰ An absolute path is the full address (like a complete GPS address).

It tells the browser exactly where the file is, starting from the root directory or a full URL.

Example :

  1. Using a full URL (web) :

    • <img src="https://example.com/images/car.jpg" alt="Car">

  2. Using a full computer path (not recommended on web):

    • <img src="C:/Users/Soham/Pictures/car.jpg" alt="Car">

When to use :

  • If the file is outside your project.

  • If you want to link to an external websiteâ€™s file (like an image or CSS).

2. Relative Path :

ðŸ‘‰ A relative path is like giving directions from your current location.

It shows where the file is in relation to the HTML fileâ€™s location.

Example :

  1. File in same folder :

    • <img src="car.jpg" alt="Car">

  2. File in a sub-folder :

    • <img src="images/car.jpg" alt="Car">

  3. File in a different folder (go back one level with ..) :

    • <img src="../car.jpg" alt="Car">

  4. Go back two levels :

    • <img src="../../car.jpg" alt="Car">

When to use :

  • For files inside your project (images, CSS, JS).

  • Makes project easy to move (because paths are relative, not fixed).
