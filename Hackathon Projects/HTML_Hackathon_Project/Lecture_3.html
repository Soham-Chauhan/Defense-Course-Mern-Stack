                                                                              What is List and Tables?

List Elements (<ul>, <ol>, <li>) :

What it is: These tags are used to create lists. This is a perfect example of nesting.

The Tags:

  • <ul>: An Unordered List. It creates a bulleted list.
  • <ol>: An Ordered List. It creates a numbered list.
  • <li>: A List Item. Each item in either type of list must be wrapped in an <li> tag.

The Purpose: To group related items together in a list format.

The <li> elements are nested as children inside the <ul> parent.

Example (Nesting in action):

Output :

Unordered List :

  • Tea Bag
  • Water
  • Sugar
  • Milk

Ordered List :

  1. First, boil the water in a kettle.
  2. Add tea bag, sugar and milk into it
  3. Keep boiling it for 5 minutes
  4. Serve the tea by pouring it into cup

Nested List :

  • A list inside another list.
  • Means you put <ul> (unordered list) or <ol> (ordered list) inside an <li> item.

Example :

Output :

Fruits and their Types

  • Apple
  • Banana
  • Mango
    • Alphonso
    • Dasheri
    • Kesar
  • Orange

Tables in HTML :

A table is used to display data in rows and columns (like in Excel).We use these tags:

We use these tags :

  • <table> → creates the table

  • <tr> → table row

  • <th> → table header (bold, centered by default)

  • <td> → table data (normal cell)

Table Example :

Output :

Name Subject Marks
Soham Math 95
Ravi Science 89
Aditi English 92

Most Usefull Table Attribute :

  1. border → Sets the table border.

    • <table border="1">

  2. align → Aligns table or content (left, right, center).

    • <table border="1" align="center">

  3. colspan → Merge columns (horizontally).

    • <td colspan="2">This cell covers 2 columns</td>

  4. rowspan → Merge rows (vertically).

    • <td rowspan="2">This cell covers 2 rows</td>
